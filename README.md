# Miniguia-finaceiro
🎯 Contexto e Objetivos

Este projeto foi desenvolvido com o objetivo de utilizar a Inteligência Artificial como ferramenta de aprendizagem ativa por meio do NotebookLM.

O tema escolhido foi Educação Financeira Básica, com foco em desenvolver conhecimentos essenciais para o controle e planejamento financeiro pessoal.

🎯 Objetivos de estudo:

Compreender os conceitos fundamentais de educação financeira

Aprender a diferenciar receitas e despesas

Desenvolver habilidades de organização financeira

Utilizar IA para estruturar e revisar conteúdos

Criar um material reutilizável para estudos futuros

⚙️ Engenharia de Prompts e Testes
🔍 Prompt 1 (Inicial – Genérico)
O que é educação financeira?

📉 Resultado: resposta ampla e pouco detalhada

🔧 Prompt 1 (Refinado)
Explique o que é educação financeira com exemplos do dia a dia

📈 Resultado: resposta mais prática e compreensível

🔍 Prompt 2
Qual a diferença entre receita e despesa?

✔️ Resultado: definição correta, porém simples

🔧 Prompt 2 (Refinado)
Explique a diferença entre receita e despesa com exemplos reais

📈 Melhoria: maior clareza e aplicação prática

🔍 Prompt 3
Como montar um orçamento mensal?

📉 Resultado: pouco detalhado

🔧 Prompt 3 (Refinado)
Crie um passo a passo para montar um orçamento mensal com categorias reais de gastos

📈 Resultado: resposta mais estruturada e útil
